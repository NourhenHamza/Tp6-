# Tp6-
Programmation Réseau avec les sockets TCP

## Exercice 1 : Communication basique Client/Serveur
- Des commentaires ont été ajoutés dans `SocketClient.java` et `SocketServeur.java` pour expliquer le fonctionnement du code.
- Un schéma des flux de messages échangés est inclus dans le code.

## Exercice 2 : Échange d'objets Voiture entre Client/Serveur
 
- La classe `Voiture` a été créée avec des méthodes pour manipuler le carburant.
- Le client envoie une instance de `Voiture` au serveur.
- Le serveur reçoit et modifie la voiture, puis la renvoie au client.

## Exercice 3 : Application distribuée Client/Serveur avec échange de données sur des personnes
 
- La classe `Personne` a été créée avec des attributs d'âge et de nom.
- Le client envoie les informations d'une personne au serveur.
- Le serveur renvoie un identificateur au client.
